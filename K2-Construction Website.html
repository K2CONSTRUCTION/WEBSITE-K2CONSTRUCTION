SOP សំណង់ពេញលេញ

នីតិវិធីប្រតិបត្តិការស្តង់ដារសម្រាប់រាល់ដំណើរការការងារសំខាន់ៗ ធានាប្រសិទ្ធភាព តម្លាភាព គុណភាព និងសុវត្ថិភាព។

  • ជ្រើសរើសបុគ្គលិក៖ ស្នើសុំ → ផ្សព្វផ្សាយ → សម្ភាសន៍ ២ដំណាក់កាល → ត្រួតពិនិត្យប្រវត្តិ → ចេញលិខិតផ្តល់ការងារ
  • ការទទួលបុគ្គលិកថ្មី៖ បណ្តុះបណ្តាលសុវត្ថិភាពជាកាតព្វកិច្ច ណែនាំគោលការណ៍ក្រុមហ៊ុន ផ្តល់ PPE
  • វត្តមាន-ប្រាក់ខែ៖ Time Sheet ប្រចាំថ្ងៃ → អនុម័តដោយប្រធានគម្រោង → គណនាប្រាក់ខែ+OT
  • វិន័យ៖ ព្រមានផ្ទាល់មាត់ → លាយលក្ខណ៍អក្សរ → បញ្ចប់ការងារ

មិនទាន់មានសំណើទេ

មិនទាន់មានទិន្នន័យទេ

ទម្រង់កំណត់ត្រាវិន័យ (Disciplinary Record)

មិនទាន់មានទិន្នន័យទេ
  • ថវិកាគម្រោង៖ បង្កើតថវិកាលម្អិត តាមដានចំណាយពិតធៀបនឹងផែនការជារៀងរាល់សប្តាហ៍
  • អនុម័តការចំណាយ៖ ក្រោម $500 = ប្រធានផ្នែក • $500-$2,000 = នាយកគម្រោង • $2,000-$10,000 = នាយកប្រតិបត្តិការ • លើស $10,000 = ថ្នាក់គ្រប់គ្រងកំពូល
  • ទូទាត់អ្នកម៉ៅការ/អ្នកផ្គត់ផ្គង់៖ ត្រួតពិនិត្យការងារ → ផ្គូផ្គងវិក្កយបត្រ → ទូទាត់តាមកិច្ចសន្យា (១៥-៣០ថ្ងៃ)
  • សាច់ប្រាក់តូច៖ កំណត់ដែនកំណត់ប្រចាំខែ ត្រូវការវិក្កយបត្រគ្រប់ការចំណាយ

មិនទាន់មានសំណើទេ

ទម្រង់ស្នើសុំទូទាត់ (Payment Request)

មិនទាន់មានទិន្នន័យទេ

ទម្រង់កត់ត្រាសាច់ប្រាក់តូច (Petty Cash)

សមតុល្យនៅសល់៖ $500.00 (ដែនកំណត់ប្រចាំខែ $500)

មិនទាន់មានទិន្នន័យទេ
  • ស្នើសុំ → ត្រួតពិនិត្យតាមគំនូរបច្ចេកទេស+ថវិកា → ស្នើសុំតម្លៃយ៉ាងតិច ៣កន្លែង (ការទិញលើស $500)
  • អនុម័តតាមកម្រិតតម្លៃ → ចេញ Purchase Order → ត្រួតពិនិត្យសម្ភារៈពេលទទួល → ចុះឃ្លាំង

ទម្រង់ស្នើសុំទិញសម្ភារៈ (Purchase Request)

បញ្ជីស្នើសុំទិញ + អនុម័តតាមកម្រិតតម្លៃ

ក្រោម $500=ប្រធានផ្នែក • $500-$2,000=នាយកគម្រោង • $2,000-$10,000=នាយកប្រតិបត្តិការ • លើស $10,000=ថ្នាក់គ្រប់គ្រងកំពូល

មិនទាន់មានសំណើទេ
  • ផែនការគម្រោង៖ កាលវិភាគលម្អិត (Gantt Chart) + កំណត់ Milestones
  • ការផ្លាស់ប្តូរ (Change Order)៖ កត់ត្រាជាលាយលក្ខណ៍អក្សរ → វាយតម្លៃផលប៉ះពាល់ → អនុម័តពីអតិថិជន+គ្រប់គ្រង មុនអនុវត្ត
  • កិច្ចសន្យា៖ រក្សាទុកជាប្រព័ន្ធកណ្តាល តាមដានកាលកំណត់សំខាន់ៗ

តារាងកាលវិភាគគម្រោង (Gantt Chart)

កំពុងអនុវត្ត Milestone មិនទាន់ចាប់ផ្តើម
